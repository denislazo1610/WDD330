  • Encapsulation:

    • The inner workings are kept hidden inside the object and only the essential functionalities are exposed to the end user. Like a blender, we just know that we need to press a button to make it work, however we do not know what is happening inside

  • Polymorphism

    • The same process can be used for different objects. There are other artefacts that have a button in order to work, just like the blender, but they may give a different output
