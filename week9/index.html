  • How to use the Optional Chaining (?.)

    • It is going to see if an object has a specific attribute without compiling an error. If there is not that attribute, then it is going to return a undefined 

    

console.log(restaurant.opening.monday?.time)
  • Short Circuiting

    • How it is really works

     
console.log('true' || false) 
// 'true' (|| will find the true value) 

console.log('yes' && 0) 
// 0 (&& will find the false value)

console.log("nope" && "no") 
// "no" (If both are true values, 
then it return the last value)
  • How to use the Nullish Coalescing Operator

    • If we use || then it is going to return the true values. However, how about if the value of a variable is 0? Using the || will not be a option. We have to use the ?? operator 

    
let guests = 0;

guests = guests ?? 10; 

console.log(guests) // 0 (?? it is going to return 
anything that is not undefined or null)
