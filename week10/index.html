  • How to for-loop an Object?

    • Well it depends on what do you want to get.

    If you want to get the entries do this.

    

for (const [number, person] of Object.entries(scores))
console.log(`Goal ${number + 1}: ${person}`);

    To get the values of an object

     
for (const value of Object.values(scores))
console.log(value);

    or

    
const entries = Object.entries(openingHours);
console.log(entries);
            
for (const [day, { open, close }] of entries) {
  console.log(`On ${day} we open at ${open} and close at ${close}`);
  // In here we have objects in objects
}
